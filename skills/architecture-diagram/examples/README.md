# Architecture Diagram Examples

Use these examples as style presets, then swap components and connections:

- `minimal-dark.html` - compact dark style for engineering notes
- `minimal-light.html` - clean light style for docs/slides
- `blueprint-grid.html` - infra/ops blueprint with strong grid and boundaries
- `editorial-cards.html` - narrative style with summary cards for reviews

Recommended workflow:

1. Pick the closest example.
2. Save as `YYYYMMDD-<slug>.html`.
3. Replace labels, components, and arrows.
4. Keep color semantics from `../references/CONVENTIONS.md`.
